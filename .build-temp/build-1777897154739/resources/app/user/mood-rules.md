# Mood → Music Rules

## Emotional Triggers

| User Says | Meaning     | Action                                        |
|-----------|-------------|-----------------------------------------------|
| "累了"     | Tired       | Ambient / no lyrics / slow BPM                |
| "开心"     | Happy       | Upbeat indie / higher BPM / major key         |
| "专注"     | Focus mode  | Instrumental post-rock / lo-fi hip-hop only   |
| "emo"     | Melancholic | Post-rock / sadcore / minor key deep cuts     |

## Environmental Triggers

| Condition      | Action                                          |
|----------------|-------------------------------------------------|
| Rain           | Rainy day playlist: melancholic but warm        |
| Snow           | Quiet, ambient, sparse                          |
| Friday evening | Energetic, weekend celebration vibe             |
| Sunday evening | Reflective, calm before the week                |
| Late night     | No lyrics, deep ambient, slow fade              |
