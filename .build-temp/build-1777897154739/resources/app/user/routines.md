# My Daily Routines

## Monday to Friday
| Time  | Context          | Music Need                              |
|-------|------------------|-----------------------------------------|
| 07:00 | Wake up          | Energetic but not aggressive            |
| 09:00 | Start work       | Instrumental, focus, no lyrics          |
| 12:00 | Lunch break      | Light, refreshing, anything goes        |
| 15:00 | Afternoon slump  | Upbeat, re-energize                     |
| 18:00 | Off work         | Relax, transition to evening mood       |
| 22:00 | Wind down        | Ambient, lo-fi, deep cuts               |

## Weekend
| Time  | Context          | Music Need                              |
|-------|------------------|-----------------------------------------|
| 10:00 | Late morning     | Lazy, warm, acoustic                    |
| 16:00 | Afternoon        | Open to discovery, new music exploration |
| 21:00 | Evening          | Longer tracks, immersive albums         |
